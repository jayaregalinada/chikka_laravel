# Laravel + Chikka

Laravel project using the `jag/chikka` package.

### Configuration
Make sure to edit your `.env` file before everything else.
```
CHIKKA_SHORTCODE=CHIKKA_SHORTCODE
CHIKKA_KEY=CHIKKA_KEY
CHIKKA_SECRET=CHIKKA_SECRET
```
